# Project Management Dashboard

## Overview
A Python-based Project Management Dashboard with Excel 
integration, data visualization, and progress tracking 
for managing multiple projects and tasks efficiently.

## Features
- Excel file reading and writing with openpyxl
- Project progress tracking and visualization
- Team member task assignment
- Deadline management and alerts
- Interactive charts and graphs
- Export reports to Excel and PDF
- Budget tracking per project
- Status updates and notifications

## Technologies Used
- Python 3.11
- Pandas for data manipulation
- OpenPyXL for Excel integration
- Matplotlib & Seaborn for visualization
- NumPy for calculations
